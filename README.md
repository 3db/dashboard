# dashboard
Dashboard to explore 3DB experiments

## Run the dasboard

### Github hosted

- Launch `python -m threedb.dashboard`
- browse: https://3db.github.io/dashboard/  and enter the url of the server started by the command above.


### Locally hosted

- Launch `python -m threedb.dashboard`
- Clone this repo
- `npm install`
- `npm run dev`
